ChessEngine.
Implemented from scratch using object oriented programming principles. Capable of generating all legal moves in any given position,
including castling, en-passant and promotion. Uses minimax to find best move. Searches upto depth of 3. Alpha-beta pruning is used
to boost performance.

NeuralNetwork.
Implemented backpropagation from scratch. Created training loop and experimented with network size, learning rate, train-test split
and other hyperparameters to obtain best performance. Weights and biases are periodically written to CSV files so progress is not lost
when training is interrupted.

TicTacToe.
Experimented with basics of reinforcement learning. Value function is slowly populated over several games. Experimented with greedy
and epsilon-greedy approaches. Performed small optimisations such as symmetry breaking between equally values options.
